# GraphQLLeanIt
This the GraphQL layer of the leanit backend

# Install dependencies
Run `npm install`

# Run development server
Run `npm run dev`

# Run Json mock server
Run `npm run json:server`

# Playground
Hit  `http://localhost:4000/graphql`
